## Игра: "Проверка на четность"
[![asciicast](https://asciinema.org/a/Eum8ZaUFsUVdtDUhdAH4NLyAM.svg)](https://asciinema.org/a/Eum8ZaUFsUVdtDUhdAH4NLyAM)
